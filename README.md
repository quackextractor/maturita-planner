# Maturita Planner

[![Version](https://img.shields.io/badge/version-1.3.4-blue.svg)](https://github.com/quackextractor/maturita-planner)
[![Python 3.8+](https://img.shields.io/badge/python-3.8+-blue.svg)](https://www.python.org/downloads/)

A visual drag-and-drop daily planner designed specifically for managing markdown-based study schedules.

## Features
* **Markdown Parsing**: Automatically reads your study blocks and daily routine slots.
* **Drag and Drop**: Drag tasks into your daily routine visually.
* **Progress Tracking**: Tick off completed tasks and save your progress.
* **Plan Preservation**: Keeps your original plan intact and allows for daily resets.

## Usage
Run the application:
```bash
python main.py
```
Place your `plan.md` and `routine.md` in the root folder. The app will generate a `data/` folder to save your daily progress.
